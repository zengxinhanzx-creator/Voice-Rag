# Security

If you believe you have found a security vulnerability, please **do not** open a public issue.

Instead, report it responsibly:

- Open a **private security advisory** on GitHub (repository **Security** tab → **Report a vulnerability**), if enabled for this repository, or
- Contact the maintainers through the channels listed in the repository README or organization profile.

Please include enough detail to reproduce or assess the issue (version, configuration, impact).
